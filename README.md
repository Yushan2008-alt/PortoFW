# Forza Studio Portfolio

Website portofolio tim Forza Studio.

## Local Development
```bash
pnpm install
cp .env.local.example .env.local
# Fill in your RESEND_API_KEY dan CONTACT_EMAIL
pnpm dev
```

## Environment Variables
| Variable | Description |
|---|---|
| RESEND_API_KEY | API key dari resend.com |
| CONTACT_EMAIL | Email tujuan pesan dari contact form |
